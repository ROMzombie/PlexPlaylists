# PlexPlaylists

## Setup
1. cp _config.json config.json 
2. nano config.json
   (populate it with your configuration settings)
3. npm install

## Running
1. nodejs .

## Rebuilding UI
1. npm install -g polymer-cli
2. bower install
3. polymer build
